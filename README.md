## AWS Outposts

### 概述

AWS Outposts 是一项完全托管的服务，可将 AWS 基础设施、AWS 服务、API 和工具扩展到几乎任何数据中心、共处空间或本地设施，以实现真正一致的混合体验。AWS Outposts 非常适合**需要低延迟访问本地系统、本地数据处理或本地数据存储的工作负载**。

### 应用

AWS Outposts 解决了多种行业中的低延迟应用程序需求和本地数据处理需求。
